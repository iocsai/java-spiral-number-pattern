# java-spiral-number-pattern
Java challenge from sololearn.com

Input any number from user and generate pattern as follow: 

    1) Input : 5 
    Output:   1  2  3  4  5 
             16 17 18 19  6 
             15 24 25 20  7 
             14 23 22 21  8 
             13 12 11 10  9 
             
Or you can make a code for this one too. 

    2) Input: 15 
    Output:  7  8  9 10 
             6  1  2 11 
             5  4  3 12 
               15 14 13 
               
Upto what user gives input. 

## Built with

 - Java SE JDK 1.8
 - NetBeans IDE 9.0
